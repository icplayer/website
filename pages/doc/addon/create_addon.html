Create addon

Addons are used to extend the functionality of the player by providing new module types. You can find source code for all addons under /addons folder.

Addons step by step:

Step 1. Create folder for your addon

Lets say that you want to create addon with the name MyAddon. First step is to create directory:
<icplayer>/addons/MyAddon

And then create src subfolder.

Step 2. Create addon files

Addon consists of:

  • addon.xml XML file which describes addon model and metadata
  • presenter.js Javascript file which implements addon functionality
  • view.html HTML view used in the player
  • preview.html HTML preview used in the editor
  • style.css CSS styles used by addon.

Step 3. Register addon in ant build system

Add task to build this addon in Addon ant build.xml

<copyAddon src="${src}" dest="${dest}" addon="AddonId"/>

Step 4. Register addon in player

To register addon in the player (and editor). Modify AddonDescriptorFactory

addDescriptor("MyAddonId", "Addon name", "Editor menu group")

Step 5. Build player

Now when you build player, your new addon will be added to player distribution
